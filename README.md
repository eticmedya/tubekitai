## Türkçe

### TubekitAI Nedir?

**TubekitAI**, yapay zeka destekli hepsi bir arada YouTube büyüme platformudur. İçerik üreticilerin kanallarını analiz etmesinden, kapak görseli üretmesine, trend keşfinden rakip analizine kadar ihtiyaç duyduğu tüm araçları tek çatı altında sunar.

### Özellikler

#### 1. Kanal Analizi
YouTube kanalınızı derinlemesine analiz edin. Performans metrikleri, kanal DNA profili ve büyüme önerileri ile kanalınızın güçlü ve zayıf yönlerini keşfedin. AI destekli önerilerle bir sonraki adımınızı planlayın.
- **Kullanılan API:** YouTube Data API v3, OpenAI GPT-4o
- **Kredi:** 1 kredi/analiz

#### 2. Video Analizi
Videolarınızın performansını detaylı şekilde ölçün. Etkileşim oranları, başlık analizi, izlenme trendleri ve optimizasyon önerileriyle her videonuzdan maksimum verim alın.
- **Kullanılan API:** YouTube Data API v3, OpenAI GPT-4o
- **Kredi:** 1 kredi/analiz

#### 3. Yorum Analizi
Binlerce yorumu saniyeler içinde AI ile analiz edin. Pozitif, negatif ve nötr duygu analizi yapın. İzleyicilerinizin önerilerini, eleştirilerini ve sorularını otomatik olarak kategorize edin.
- **Kullanılan API:** YouTube Data API v3, OpenAI GPT-4o
- **Kredi:** 3 kredi/analiz

#### 4. Kapak AI (Thumbnail)
- **Analiz:** Mevcut kapak görsellerinizi AI ile puanlayın. Okunabilirlik, kontrast, yüz görünürlüğü, duygu ve kompozisyon skorlarıyla CTR tahmini alın. YouTube linki yapıştırarak doğrudan kapak analizi yapabilirsiniz.
- **Oluşturma:** Fal.ai Nano Banana Pro modeli ile profesyonel YouTube kapak görselleri üretin. Kendi fotoğrafınızı veya ürün görselinizi referans olarak yükleyin - AI yüzünüzü birebir koruyarak profesyonel arka plan ve sinematik ışıklandırma ekler. 16:9, 9:16 ve 1:1 formatlarını destekler.
- **Kullanılan API:** OpenAI GPT-4o (analiz), Fal.ai Nano Banana Pro (üretim)
- **Kredi:** 0.5 kredi/analiz, 1 kredi/üretim

#### 5. Niş Analizi
İlgi alanlarınız, yetenekleriniz ve yaşam tarzınıza göre size en uygun YouTube nişlerini keşfedin. Düşük rekabetli, yüksek potansiyelli niş önerileri.
- **Kullanılan API:** OpenAI GPT-4o
- **Kredi:** 2 kredi/analiz

#### 6. Video Fikir Üretici
Konuya, kanalınıza veya güncel trendlere göre viral potansiyeli yüksek video fikirleri üretin. Her fikir için hedef kitle, önerilen video süresi ve içerik özeti.
- **Kullanılan API:** OpenAI GPT-4o
- **Kredi:** 1 kredi/üretim

#### 7. Trend Keşfi
Türkiye, ABD, Almanya ve birçok bölgeden yükselen trendleri ve popüler konuları keşfedin. Günlük, haftalık ve aylık trend verilerini takip edin.
- **Kullanılan API:** YouTube Data API v3
- **Kredi:** 1 kredi/keşif

#### 8. Rakip Analizi
Rakip kanalları detaylı analiz edin. İçerik stratejilerini, yüklenme düzenlerini, en başarılı videolarını ve büyüme formüllerini öğrenin.
- **Kullanılan API:** YouTube Data API v3, OpenAI GPT-4o
- **Kredi:** 3 kredi/analiz

#### 9. Anahtar Kelime Trendleri
YouTube'da yükselen anahtar kelimeleri bulun. Arama hacmi, rekabet skoru ve trend analizi ile SEO stratejinizi güçlendirin.
- **Kullanılan API:** YouTube Data API v3
- **Kredi:** 1 kredi/arama

#### 10. TransFlow (Çeviri)
Video başlıklarını, açıklamalarını, etiketlerini ve altyazıları 100+ dile SEO uyumlu şekilde çevirin. Global izleyici kitlenize ulaşın.
- **Kullanılan API:** OpenAI GPT-4o
- **Kredi:** 1 kredi/çeviri

#### 11. İçerik Üretici Okulu
Sıfırdan YouTube kanalı açma rehberi, ekipman tavsiyeleri, ilk 10 video kuralı ve motivasyon sistemi.
- **Kredi:** Ücretsiz

### API Anahtarı Alma Rehberi

<details>
<summary><strong>YouTube Data API v3</strong></summary>

1. [Google Cloud Console](https://console.cloud.google.com/) adresine gidin
2. Yeni proje oluşturun veya mevcut bir projeyi seçin
3. **API & Services > Library** bölümüne gidin
4. "YouTube Data API v3" arayın ve **Enable** yapın
5. **API & Services > Credentials** bölümüne gidin
6. **Create Credentials > API Key** seçin
7. Oluşturulan anahtarı `.env` dosyasındaki `YOUTUBE_API_KEY` alanına yapıştırın
</details>

<details>
<summary><strong>OpenAI API</strong></summary>

1. [OpenAI Platform](https://platform.openai.com/) adresine kayıt olun
2. **API Keys** bölümüne gidin
3. **Create new secret key** butonuna tıklayın
4. Oluşturulan anahtarı `.env` dosyasındaki `OPENAI_API_KEY` alanına yapıştırın
5. **Not:** GPT-4o kullanımı için ücretli plan gerekebilir
</details>

<details>
<summary><strong>Fal.ai API</strong></summary>

1. [Fal.ai](https://fal.ai/) adresine kayıt olun
2. **Dashboard > Keys** bölümüne gidin
3. Yeni API key oluşturun
4. Oluşturulan anahtarı `.env` dosyasındaki `FAL_API_KEY` alanına yapıştırın
5. **Not:** Nano Banana Pro modeli kullanılır (text-to-image ve image editing)
</details>

<details>
<summary><strong>PayTR (Ödeme)</strong></summary>

1. [PayTR](https://www.paytr.com/) adresine başvurun
2. Onaylandıktan sonra **Mağaza Ayarları** bölümüne gidin
3. Merchant ID, Key ve Salt bilgilerini `.env` dosyasına ekleyin
4. Test modu için `PAYTR_TEST_MODE=true` yapın
</details>

## English

### What is TubekitAI?

**TubekitAI** is an all-in-one AI-powered YouTube growth platform. It provides content creators with all the tools they need - from channel analysis to cover image generation, trend discovery to competitor analysis - all under one roof.

### Features

#### 1. Channel Analysis
Deep-dive analysis of your YouTube channel. Performance metrics, channel DNA profile, and growth recommendations to discover your channel's strengths and weaknesses.
- **API Used:** YouTube Data API v3, OpenAI GPT-4o
- **Credit:** 1 credit/analysis

#### 2. Video Analysis
Measure your videos' performance in detail. Engagement rates, title analysis, view trends, and optimization suggestions to maximize output from each video.
- **API Used:** YouTube Data API v3, OpenAI GPT-4o
- **Credit:** 1 credit/analysis

#### 3. Comment Analysis
Analyze thousands of comments with AI in seconds. Perform positive, negative, and neutral sentiment analysis. Automatically categorize viewer suggestions, criticisms, and questions.
- **API Used:** YouTube Data API v3, OpenAI GPT-4o
- **Credit:** 3 credits/analysis

#### 4. Cover AI (Thumbnail)
- **Analysis:** Score your existing thumbnails with AI. Get CTR predictions with readability, contrast, face visibility, emotion, and composition scores. Paste a YouTube link for direct analysis.
- **Generation:** Generate professional YouTube thumbnails with Fal.ai Nano Banana Pro model. Upload your photo or product image as reference - AI preserves your face exactly while adding professional backgrounds and cinematic lighting. Supports 16:9, 9:16, and 1:1 formats.
- **API Used:** OpenAI GPT-4o (analysis), Fal.ai Nano Banana Pro (generation)
- **Credit:** 0.5 credit/analysis, 1 credit/generation

#### 5. Niche Analysis
Discover the best YouTube niches for you based on your interests, skills, and lifestyle. Low-competition, high-potential niche recommendations.
- **API Used:** OpenAI GPT-4o
- **Credit:** 2 credits/analysis

#### 6. Video Idea Generator
Generate video ideas with high viral potential based on topic, channel, or current trends. Each idea comes with target audience, suggested duration, and content outline.
- **API Used:** OpenAI GPT-4o
- **Credit:** 1 credit/generation

#### 7. Trend Discovery
Discover rising trends and popular topics from Turkey, USA, Germany, and many other regions.
- **API Used:** YouTube Data API v3
- **Credit:** 1 credit/discovery

#### 8. Competitor Analysis
Analyze competitor channels in detail. Learn their content strategies, upload patterns, most successful videos, and growth formulas.
- **API Used:** YouTube Data API v3, OpenAI GPT-4o
- **Credit:** 3 credits/analysis

#### 9. Keyword Trends
Find rising keywords on YouTube. Strengthen your SEO strategy with search volume, competition score, and trend analysis.
- **API Used:** YouTube Data API v3
- **Credit:** 1 credit/search

#### 10. TransFlow (Translation)
Translate video titles, descriptions, tags, and subtitles to 100+ languages with SEO optimization. Reach your global audience.
- **API Used:** OpenAI GPT-4o
- **Credit:** 1 credit/translation

#### 11. Creator School
YouTube channel creation guide from scratch, equipment recommendations, first 10 videos rule, and motivation system.
- **Credit:** Free

## Kurulum / Installation

### Gereksinimler / Requirements

- PHP 8.2+
- Composer
- MySQL veya SQLite
- Node.js & NPM

### Adımlar / Steps

```bash
# 1. Repoyu klonlayın / Clone the repo
git clone https://github.com/aykutssert/tubekitai.git
cd tubekitai

# 2. Composer paketlerini yükleyin / Install Composer packages
composer install

# 3. .env dosyasını oluşturun / Create .env file
cp .env.example .env

# 4. Uygulama anahtarı oluşturun / Generate application key
php artisan key:generate

# 5. .env dosyasını düzenleyin / Edit .env file
# API anahtarlarınızı ve veritabanı bilgilerinizi girin
# Enter your API keys and database credentials

# 6. Veritabanını oluşturun / Create database
php artisan migrate

# 7. Demo verilerini yükleyin / Seed demo data
php artisan db:seed

# 8. Storage linkini oluşturun / Create storage link
php artisan storage:link

# 9. Sunucuyu başlatın / Start the server
php artisan serve
```

Tarayıcınızda `http://localhost:8000` adresine gidin.

### .env Yapılandırması / .env Configuration

```env
# YouTube API - https://console.cloud.google.com/
YOUTUBE_API_KEY=your-youtube-api-key

# OpenAI - https://platform.openai.com/api-keys
OPENAI_API_KEY=your-openai-api-key

# Fal.ai - https://fal.ai/dashboard/keys
FAL_API_KEY=your-fal-ai-api-key

# PayTR (isteğe bağlı / optional) - https://www.paytr.com/
PAYTR_MERCHANT_ID=your-merchant-id
PAYTR_MERCHANT_KEY=your-merchant-key
PAYTR_MERCHANT_SALT=your-merchant-salt
```
